Router

The router is responsible for handling HTTP requests and directing them to the appropriate controller actions.

GET Router::get(string $path, mixed $handler)

Register a new GET route with the router.

Parameters

Name Type Description
$path string The URL pattern to match
$handler mixed Controller action or closure

Example

// Basic route
$router->get('/', 'HomeController@index');

// Route with parameters
$router->get('/users/{id}', 'UserController@show');

// Route with closure
$router->get('/hello', function() {
    return 'Hello World';
});
